Hosting HTML Static Secara Gratis di Google Cloud

Master
Category :hosting Tags: hosting, cloud, static, 12/8/2024

Berikut adalah langkah-langkah membuat dan menghosting HTML static di Google Cloud secara gratis, sekaligus mengimplementasikan form untuk mencatat guestbook:

1. Setup Google Cloud Free Tier

  1. Daftar Akun Google Cloud:

    • Jika belum punya akun, daftar di Google Cloud.
    • Google Cloud menawarkan $300 kredit gratis selama 12 bulan, serta layanan tertentu gratis selamanya (termasuk hosting di Firebase).

  2. Aktifkan Firebase Hosting:

    • Firebase adalah layanan Google Cloud untuk hosting web statis secara gratis dalam batas wajar.

2. Buat Project Firebase

  1. Buka Firebase Console: Firebase Console.

  2. Buat Project Baru:

    • Klik Add Project.
    • Beri nama proyek, misalnya guestbook-static.
    • Ikuti langkah hingga proyek selesai dibuat.

  3. Aktifkan Hosting:

    • Di dashboard proyek, pilih Build > Hosting.
    • Klik Get Started.

3. Persiapkan HTML Static

  1. Buat File HTML Static:

    • Buat file bernama index.html di komputer Anda. Berikut contoh HTML sederhana dengan form untuk mencatat guestbook:
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Guestbook</title>
</head>
<body>
    <h1>Welcome to My Guestbook</h1>
    <form id="guestbook-form">
        <input type="text" id="name" placeholder="Your Name" required>
        <textarea id="message" placeholder="Your Message" required></textarea>
        <button type="submit">Submit</button>
    </form>
    <ul id="guestbook-list"></ul>
    <script>
        const form = document.getElementById('guestbook-form');
        const list = document.getElementById('guestbook-list');

        form.addEventListener('submit', async (e) => {
            e.preventDefault();
            const name = document.getElementById('name').value;
            const message = document.getElementById('message').value;

            await fetch('https://your-project-id.firebaseio.com/guestbook.json', {
                method: 'POST',
                headers: { 'Content-Type': 'application/json' },
                body: JSON.stringify({ name, message, date: new Date().toISOString() })
            });

            const listItem = document.createElement('li');
            listItem.textContent = `${name}: ${message}`;
            list.appendChild(listItem);

            form.reset();
        });
    </script>
</body>
</html>

4. Host di Firebase

  1. Install Firebase CLI:

    • Instal Firebase CLI di komputer Anda.
    • Jalankan:
      npm install -g firebase-tools

  2. Login ke Firebase:

    firebase login

  3. Inisialisasi Proyek:

    • Di direktori tempat index.html berada, jalankan:
      firebase init
    • Pilih Hosting > Proyek yang telah dibuat > Public Directory isikan . > Yes untuk SPA.

  4. Deploy:

    • Jalankan:
      firebase deploy
    • URL hosting akan diberikan, misalnya: https://guestbook-static.web.app.

5. Catat Data di Firebase Realtime Database

  1. Aktifkan Realtime Database:

    • Di Firebase Console, pilih Build > Realtime Database.
    • Klik Create Database > Pilih lokasi > Set mode ke test mode (untuk testing).

  2. Ganti Endpoint di JavaScript:

    • Update URL di script:
      await fetch('https://your-project-id.firebaseio.com/guestbook.json', {

6. Domain Name Custom (Opsional)

Jika punya domain, Anda bisa menghubungkannya ke Firebase Hosting:

  1. Buka Hosting di Firebase Console.
  2. Tambahkan domain custom dan verifikasi melalui DNS.

Hasilnya, Anda akan memiliki website HTML static gratis dengan fitur guestbook yang mencatat data di Firebase Realtime Database.
