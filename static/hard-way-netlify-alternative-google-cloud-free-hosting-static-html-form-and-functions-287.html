Menggunakan Google Cloud untuk Mendapatkan Fasilitas Free Hosting Static HTML, Form Submission, dan Serverless Function

Langkah 6: Membuat Subdomain menggunakan Google Cloud DNS

  • Buat subdomain menggunakan Google Cloud DNS.
  • Pastikan Anda memiliki akses ke subdomain tersebut.

Langkah 7: Menggunakan HTTPS menggunakan Google Cloud SSL

Langkah 9: Membuat Serverless Function menggunakan Google Cloud Functions

  • Buat fungsi serverless menggunakan Google Cloud Functions.

Langkah 10: Mengintegrasikan Semua Layanan

  • Integrasikan semua layanan yang telah Anda buat untuk mendapatkan fasilitas free hosting static HTML, form submission, dan serverless function.

Use Custom Own Domain in Google Cloud ?

Menggunakan Custom Own Domain di Google Cloud

Ya, di Google Cloud Anda dapat menggunakan custom own domain untuk mengakses aplikasi atau layanan yang Anda buat. Berikut adalah beberapa cara untuk menggunakan custom own domain di Google Cloud:

  2. Menggunakan Google Cloud Load Balancing: Anda dapat menggunakan Google Cloud Load Balancing untuk mengatur load balancer yang dapat mengarahkan lalu lintas ke aplikasi atau layanan yang Anda buat di Google Cloud. Dengan demikian, Anda dapat menggunakan custom own domain untuk mengakses aplikasi atau layanan yang Anda buat.

Newbie Tutorial: Using Google Cloud for Free Hosting of Static HTML, Form Submission, and Serverless Function

Step 1: Create a Google Cloud Account

  • Go to the Google Cloud website (https://cloud.google.com) and click on "Get started" in the top right corner.
  • Follow the prompts to create a new Google Cloud account.
  • Verify your email address and set up a password.

Step 2: Create a New Project

  • Log in to the Google Cloud Console (https://console.cloud.google.com).
  • Click on the "Select a project" dropdown menu and click on "New Project".
  • Enter a project name and click on "Create".

Step 3: Enable the Cloud Storage API

  • In the sidebar, click on "APIs & Services" and then click on "Dashboard".
  • Click on "Enable APIs and Services" and search for "Cloud Storage".
  • Click on "Cloud Storage API" and click on "Enable".

Step 4: Create a New Bucket

  • In the sidebar, click on "Storage" and then click on "Buckets".
  • Click on "Create bucket" and enter a bucket name.
  • Click on "Create".

Step 5: Upload Your Static HTML Files

  • In the sidebar, click on "Storage" and then click on "Buckets".
  • Click on the bucket you just created and click on "Upload files".
  • Select the static HTML files you want to upload and click on "Open".

Step 6: Create a New Serverless Function

  • In the sidebar, click on "Cloud Functions" and then click on "Create function".
  • Enter a function name and select "HTTP" as the trigger.
  • Click on "Create".

Step 7: Configure the Serverless Function

  • In the function code editor, paste the following code:
from flask import Flask, request

app = Flask(__name__)

@app.route('/', methods=['GET', 'POST'])
def handle_form_submission():
    # Handle form submission logic here
    return 'Form submitted successfully!'
  • Click on "Deploy" to deploy the function.

Step 8: Configure Form Submission

  • In the sidebar, click on "Cloud Functions" and then click on the function you just created.
  • Click on "Trigger" and select "HTTP" as the trigger.
  • Enter the URL of your static HTML file and click on "Save".

Step 9: Test Your Application

  • Go to the URL of your static HTML file and fill out the form.
  • Submit the form and verify that the serverless function is triggered correctly.

Congratulations! You have successfully set up a free hosting of static HTML, form submission, and serverless function using Google Cloud.
